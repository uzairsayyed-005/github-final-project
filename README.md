# Simple Interest Calculator

This project is a simple web-based application that calculates simple interest based on user input.

## 📌 Features
- Enter principal amount
- Enter rate of interest
- Enter time period
- Calculates simple interest instantly

## 🧮 Formula
Simple Interest = (Principal × Rate × Time) / 100

## 🚀 Usage
1. Enter the required values
2. Click on Calculate
3. View the result

## 🛠️ Technologies Used
- HTML
- JavaScript
